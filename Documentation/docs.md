#Palette
###Objects
#####Color
**class** color:
  __r__: number = Red value of the color
  __g__: number = Green value of the color
  __b__: number = Blue value of the color

  __render__: string = CSS rgb() function of the color

**const** colors:
`Basic, preset colors to use with Palette. Includes red, green, blue, white, black, gray, yellow, pink, and purple`
